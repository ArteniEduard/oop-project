#  **QuickDeliver**

## **Overview**
**QuickDeliver** is a console-based delivery management system inspired by modern food delivery platforms. It models the interaction between **restaurants**, **couriers**, and **orders** using object-oriented programming principles.

## **Purpose**
The application simulates the lifecycle of an order, from **creation** to **delivery**, focusing on:
- clean design  
- data encapsulation  
- structured interactions between system components  

## **RUN**
mkdir build
cd build
cmake ..
cmake --build .
